Mi trayectoria

"Si piensas como un programador, puedes lograrlo"

Empecé en esto de la programación cuando era más joven, con cosas simples. No se le podría llamar programación, pero la primera aplicación que hice, fue con C#, viendo muchos tutoriales para hacer la parte visual y buscando códigos en internet, aunque no lograba entenderlos. Cuando hice el curso de confección y publicación de páginas web de Labora, empecé a entender cómo funcionaba todo. Por mi cuenta, empecé a aprender Python y Django, porque hacer páginas web me llamó mucho la atención. Aunque también quería aprender a hacer aplicaciones de escritorio o aplicaciones móviles, vi que Rust era mucho mejor para crear ejecutables, y encima también sirve para hacer webs de alto rendimiento. Actualmente, estoy haciendo otro curso de Labora de desarrollo de aplicaciones web.
